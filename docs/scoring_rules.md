# 评分规则

本项目使用 100 分制对候选商品进行排序。评分不是检测结论，而是基于公开信息的合规初筛推荐规则。

## 总分结构

| 评分项 | 权重 |
|---|---:|
| 标准匹配度 | 40 分 |
| 证据可信度 | 25 分 |
| 用户需求匹配度 | 15 分 |
| 价格合理性 | 10 分 |
| 品牌/渠道可信度 | 10 分 |

## 标准匹配度：40 分

| 得分区间 | 判断标准 |
|---|---|
| 35-40 | 商品明确标注相关标准，且标准与产品品类高度匹配 |
| 25-34 | 商品有相关标准说明，但适用范围仍需二次确认 |
| 10-24 | 只有泛化安全描述，没有明确标准编号 |
| 0-9 | 没有任何相关标准信息 |

## 证据可信度：25 分

| 得分区间 | 判断标准 |
|---|---|
| 21-25 | 来自政府标准平台、品牌官网或官方检测报告 |
| 15-20 | 来自官方旗舰店或电商详情页，信息较完整 |
| 5-14 | 来自第三方文章或信息不完整 |
| 0-4 | 无公开证据 |

## 用户需求匹配度：15 分

| 得分区间 | 判断标准 |
|---|---|
| 13-15 | 同时满足使用场景、预算、材质、安全和特殊要求 |
| 9-12 | 满足主要需求，但个别要求缺少证据 |
| 4-8 | 只满足部分需求，存在明显信息缺口 |
| 0-3 | 与用户场景或核心要求不匹配 |

## 价格合理性：10 分

| 得分区间 | 判断标准 |
|---|---|
| 8-10 | 价格在预算内，且与同类产品价格相对合理 |
| 5-7 | 略高于预算或价格信息不完整 |
| 1-4 | 明显超预算，或价格与证据质量不匹配 |
| 0 | 未找到公开价格且无法判断 |

## 品牌/渠道可信度：10 分

| 得分区间 | 判断标准 |
|---|---|
| 8-10 | 品牌官网、官方旗舰店或授权渠道信息清晰 |
| 5-7 | 有品牌信息，但官方来源不足 |
| 1-4 | 品牌信息模糊，主要来自第三方页面 |
| 0 | 无法识别品牌或渠道 |

## 推荐等级

| 总分 | 推荐等级 | 解释 |
|---:|---|---|
| 85-100 | 强推荐 | 标准和证据链较完整 |
| 70-84 | 可推荐 | 基本满足需求，但仍需购买前确认 |
| 50-69 | 谨慎推荐 | 证据不足或需求匹配一般 |
| 0-49 | 不推荐 | 缺少关键证据或风险较高 |

## 强制降级规则

| 情况 | 处理方式 |
|---|---|
| 没有商品层面证据 | 最高只能评为“谨慎推荐” |
| 商品链接缺失 | 不输出购买推荐，只输出信息线索 |
| 只有第三方文章证据 | 最高只能评为“谨慎推荐” |
| 标准编号无法确认 | 不允许写“符合标准” |
| 涉及儿童、食品接触、电器等高风险品类 | 证据不足时必须降级 |
