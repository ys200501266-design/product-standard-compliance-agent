# 评分规则

## 权重

| 维度 | 权重 | 评分说明 |
|---|---:|---|
| 标准匹配度 | 40% | 商品是否公开标注相关标准、认证或检测信息 |
| 信息可信度 | 20% | 证据是否来自官方平台、品牌官网、官方旗舰店或检测报告 |
| 价格合理性 | 15% | 是否符合预算，是否明显偏离同类价格 |
| 品牌可信度 | 15% | 是否有可验证品牌官网、官方店铺或稳定公开信息 |
| 用户需求匹配度 | 10% | 是否满足场景、材质、防漏、容量、儿童使用等要求 |

## 推荐原则

1. 有明确公开证据的商品优先。
2. 品牌官网和检测报告优先于第三方软文。
3. 电商标题中的标准信息只能作为弱证据，需要详情页或图片佐证。
4. 缺少证据的商品不能写成“符合标准”。
5. 如果所有候选商品证据不足，应输出风险提示，不做确定性推荐。
