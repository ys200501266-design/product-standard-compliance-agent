# 风险与免责声明

## 风险边界

本项目用于公开信息初筛，不提供法律意见、检测结论或官方认证结论。模型可能受搜索结果覆盖范围、网页摘要质量、动态页面抓取能力和标准适用性判断影响。

## 必须保留的表达

当没有找到官方或页面证据时，必须写：

```text
暂无公开证据证明
```

当商品只有营销词时，应写：

```text
页面仅出现营销描述，未看到明确执行标准、认证或检测报告。
```

## 购买前建议

- 查看商品详情页是否标注执行标准。
- 查看包装或标签照片。
- 向品牌客服索要检测报告或材质说明。
- 优先选择品牌官网、官方旗舰店或监管平台可验证的信息。

## 免责声明

以上结果基于公开信息整理，不构成强制检测结论。产品是否完全符合相关标准，应以品牌官方说明、检测报告、商品包装标识和监管机构信息为准。
