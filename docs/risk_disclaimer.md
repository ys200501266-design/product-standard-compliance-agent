# 风险控制与免责声明

## 风险边界

本项目用于公开信息初筛，不提供法律意见、检测结论或官方认证结论。模型输出受搜索结果覆盖范围、网页摘要质量、动态页面抓取能力和标准适用性判断影响。

## 主要风险

| 风险 | 说明 | 控制方式 |
|---|---|---|
| 标准误用 | 某标准真实存在，但不一定适用于该商品 | 输出适用范围和不确定性 |
| 链接缺失 | 搜索 API 未返回真实商品链接 | 不编造链接 |
| 电商宣传误导 | 页面营销词可能没有检测依据 | 降低证据等级 |
| LLM 幻觉 | 模型可能补全标准或报告 | Prompt 禁止编造 |
| 搜索失败 | API 不稳定或关键词过窄 | 返回兜底报告 |

## 必须保留的表达

当没有找到商品层面证据时，必须写：

```text
暂无公开证据证明该产品完全符合该标准。
```

当搜索结果不足时，必须写：

```text
未找到足够的公开标准信息，本次结果仅能作为初步搜索线索，不构成推荐结论。
```

当商品证据不足时，必须写：

```text
未找到足够的商品公开证据，因此不输出强推荐商品。
```

## 免责声明

以上结果基于公开信息整理，不构成强制检测结论、法律意见或官方认证结论。产品是否完全符合相关标准，应以品牌官方说明、检测报告、商品包装标识和监管机构信息为准。
