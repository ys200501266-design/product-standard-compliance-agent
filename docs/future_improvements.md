# 未来优化方向

## 数据能力

| 方向 | 说明 |
|---|---|
| 标准知识库 | 缓存常见品类标准、适用范围和状态 |
| 证据截图 | 保留搜索结果和详情页截图 |
| 时间戳 | 记录证据抓取时间，便于复核 |

## 检索能力

| 方向 | 说明 |
|---|---|
| 网页正文抓取 | 获取商品详情页完整内容 |
| OCR | 识别详情图中的执行标准和检测报告 |
| 多搜索源融合 | 同时接入 SerpAPI、Tavily、Firecrawl |

## 产品能力

| 方向 | 说明 |
|---|---|
| PDF 报告 | 生成采购初筛报告 |
| Notion/飞书同步 | 输出到团队协作文档 |
| 多商品对比 | 支持用户指定商品链接对比 |
| 人工审核 | 高风险品类加入人工确认节点 |

## 风控能力

| 方向 | 说明 |
|---|---|
| 高风险品类规则 | 儿童、食品接触、电器、医疗类提高证据门槛 |
| 强制降级 | 缺少商品证据时限制推荐等级 |
| 敏感词处理 | 避免“绝对安全”“一定符合”等表达 |
