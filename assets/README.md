# Assets 说明

这里用于放置真实运行截图。当前项目不生成假截图，所有截图应来自你实际运行 n8n 和 Webhook 后的结果。

## 建议补充截图

| 截图 | 作用 |
|---|---|
| workflow_screenshot.png | 证明 n8n 工作流已经搭建 |
| webhook_test.png | 证明接口可以被调用 |
| search_result_debug.png | 证明搜索节点返回真实数据 |
| final_report.png | 证明最终报告可读 |
| github_readme_preview.png | 证明项目适合 GitHub 展示 |

## 截图建议

1. 截图要包含浏览器或 n8n 页面主体，不要只截局部。
2. 如果截图里有 API Key，必须打码。
3. final_report.png 应展示表格、证据等级和风险提醒。
4. github_readme_preview.png 应展示 README 第一屏。
