# Demo Case：儿童保温杯

## 输入

```json
{
  "product_name": "儿童保温杯",
  "use_case": "小学生日常上学使用",
  "budget": "100元以内",
  "region": "中国大陆",
  "extra_requirements": "安全、防漏、材质合规、有执行标准说明"
}
```

## 预期工作流行为

1. 识别为儿童食品接触用保温杯/不锈钢真空杯场景。
2. 优先检索食品接触材料、不锈钢真空杯、真空保温容器相关标准。
3. 生成商品搜索 query，例如“儿童保温杯 执行标准 GB 4806.9 检测报告 品牌官网 官方旗舰店”。
4. 检查商品页面是否公开标注执行标准、认证或检测报告。
5. 对缺少证据的商品标注“暂无公开证据证明”。
6. 输出结构化推荐报告。

## Demo 注意事项

- 不要在 Demo 中手动伪造真实电商链接。
- 可以使用 `https://example.com/product-a` 作为占位符，并明确说明真实运行时由搜索 API 返回。
- 如果搜索 API 没有返回足够证据，报告应倾向于风险提示，而不是强行推荐。
